# SCGoldenSpear
Minecraft 1.21.11 plugin to restore saturation when using a special spear.

## Summary
Restores 3 saturation points to any player who lunges with a golden spear named `§6SweptCraft Golden Spear`.  
This allows players to lunge indefinitely without exhausting themselves.

## Motivation
My server's golden birthday is coming up soon (13 on the 13th!)  
so I'm giving away a fancy golden spear, since it's the newest weapon.  
In order to allow players to lunge as many times as they want,  
I wrote this plugin to restore the saturation a lunge consumes.

## Usage
Hold a spear named `§6SweptCraft Golden Spear` and lunge.  
That's it!  

## Installation
Check the Releases tab on the right and download the JAR file.  
Place the plugin file in your `plugins` folder.  
Restart your server.  

## Building
Add the project files to your favorite IDE.  
Build with Maven.
